## Разработка интерфейса
# Цель: создать одностраничное веб-приложение (SPA) для визуализации обращений граждан 
# Описание задачи
Разработать клиентское приложение, которое загружает список обращений из файла или API, отображает их в таблице с фильтрами, показывает данные на карте и предоставляет детальную карточку каждого обращения.
